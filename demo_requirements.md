# Project Requirement: Smart Escrow User Dashboard

## Functional Requirements
1. **Glassmorphic Header**: Must include a navigation bar with a translucent background and backdrop-blur effects.
2. **Web3 Integration**: The dashboard must display the user's connected wallet address and current network status.
3. **Responsive Grid**: Services should be displayed in a 3-column grid on desktop and a single column on mobile.
4. **AI Assistant Panel**: A dedicated sidebar area for interacting with the Gemini AI model.
5. **Real-time Status Symbols**: Visual indicators (glowing lights) for backend and blockchain connectivity.

## Technical Stack
- Framework: React (TypeScript)
- Styling: Tailwind CSS
- Icons: Lucide-React
- Blockchain Library: Ethers.js (v6)

## Acceptance Criteria
- Code matches the 90% of the UI design tokens defined in the Glassmorphism spec.
- No unused imports or variables in the final build.
- Contract interactions are handled via asynchronous hooks.
